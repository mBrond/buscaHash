Criar um programa que permite algumas buscas de dados na base de dados Empresas0 do Cadastro Nacional de Pessoa Jurídica do governo brasileiro..

No link abaixo você irá encontrar os dados:
https://dados.gov.br/dados/conjuntos-dados/cadastro-nacional-da-pessoa-juridica---cnpj
no mesmo link está disponível o dicionário de dados que explica as colunas presentes nos arquivos csv.

Seu programa deverá permitir, ao menos, as seguintes buscas de dados.
- procurar um CNPJ específico (primeira coluna do arquivo csv) e mostrar dados
- procurar por nome (segunda coluna do arquivo csv) e mostrar dados

O mecanismo de busca deve utilizar alguma das estruturas vistas na disciplina. Mais de uma estrutura/técnica de busca pode ser usada.

OBSERVAÇÕES:

- somente serão aceitos programas em C ANSI (gcc -ansi)
- grupos de, no máximo, 2 alunos
- NÃO usar pesquisa sequencial simples (somente em casos internos, com poucos elementos, a mesma pode ser usada)

Fase Bonus:
Faça um programa capaz de comprimir/descomprimir o arquivo csv utilizando algumas das técnicas vistas. Seu objetivo é conseguir uma taxa de compressão de ao menos 0,30 (para o arquivo Empresas0 o compressor zip consegue uma taxa de ~0,25)
